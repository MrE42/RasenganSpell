# Rasengan Spell

A custom spell mod for **Mage Arena** that lets you unleash Naruto’s iconic **Rasengan** (and eventually more).

## Requirements
- **BepInExPack_MageArena**
- **BlackMagicAPI**
- **FishUtilities**

(Install via Thunderstore Mod Manager.)
